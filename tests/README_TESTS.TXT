TestCoordinates.sofea
---------------------

Simple coordinate transformation test based on the AERMAP test case 
`NW_Durham/aermap_nad83_ned.inp`. The DOMAIN source should have 
latitude/longitude corresponding to the AERMAP output file 
`aermap_nad83_ned_MAPDETAIL.OUT`.

  SW corner coordinates:      36.041761    -78.963057 decimal deg.
